A) Bootstrap HTML Structure - top

This theme was built with bootstrap

let's see how to write the code:

DOCTYPE

<!DOCTYPE html>
<html lang="en">
...
</html>

INCLUDE META, CSS and JAVASCRIPT

<!DOCTYPE html>
<html>
  <head>
    <title>Preciso</title>
    <!-- For Devices -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet" media="screen">
    <!-- Preciso style -->
    <link href="css/style.css" rel="stylesheet" media="screen">
</head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

CONTAINER and GRID

Create a .row and add the appropriate number of .span* columns. As this is a 12-column grid, each .span* spans a number of those 12 columns, and should always add up to 12 for each row.

<div class="container">
  <div class="row">
    <div class="span9">
      <!--Body content-->
    </div>
    <div class="span3">
      <!--Sidebar content-->
    </div>
  </div>
</div>

B) CSS Files - top

I'm using 8 CSS files in this theme. The first two is bootstrap file.

The third file contains all of the specific stylings for the page.

we include the css file in the HEAD section of the HTML page like this:

<!-- Bootstrap CSS -->
<link href="css/bootstrap.min.css" rel="stylesheet" type="text/css" media="screen" />
<link href="css/bootstrap-responsive.min.css" rel="stylesheet" type="text/css" />

<!-- Font Awsome CSS -->
<link href="css/font-awesome.min.css" rel="stylesheet" type="text/css" media="screen" />

<!-- Preciso CSS -->
<link href="css/style.css" rel="stylesheet" type="text/css" media="screen" />

<!-- flexslider Slider CSS -->
<link href="css/flexslider.css" rel="stylesheet" type="text/css" media="screen" />

<!-- Masonry CSS -->
<link href="css/masonry.css" rel="stylesheet" type="text/css" media="screen" />

<!-- Fancybox CSS -->
<link href="css/fancybox.css" rel="stylesheet" type="text/css" media="screen" />

<!-- Custom Color CSS -->
<link href="css/color/multicolor.css" rel="stylesheet" type="text/css" id="colorLink" />

<!-- Google Font CSS -->
<link href="http://fonts.googleapis.com/css?family=Roboto+Condensed" />

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
